Bubblegum Bandit

Tools and Skills: Java, libgdx, Game Development

Embody a space pirate who wreaks havoc by flipping gravity and sticking enemies with bubblegum!
Bubblegum Bandit is an action platformer that I created alongside seven other team members over the span of three months. I was responsible for coding much of the physics and player mechanics of the game — most notably, all of the gum mechanics (how it shoots, sticks, and feels) were implemented by me. It was coded in Java using the libgdx library, and is the project I am most proud of — not only because it's my most complex, but it's simply super fun to play!

Plant Pals

Tools and Skills: Python, Javascript, HTML/CSS

Everyone Deserves a Plant
Plant Pals operates as a search engine that allows users to find their ideal indoor plant depending on the characteristics they search (ie has flowers, little watering, etc.). The backend utilized text querying concepts such as cosine similarity and jaccard to match the user's typed input to a plant's description. Additionally, Rocchio's algorithm was used so that users could improve their search results depending on whether they liked the ones they were given.

Climate Fund App

Tools and Skills: React Native, Team Leadership, Organization

This app is a project I did with my project team, Engineers Without Borders. It was in collaboration with the Thompkins Finger Lakes Climate Fund. Using the app, users can offset their carbon emissions by donating a corresponding amount of money to the Climate Fund, which uses the money to support local green projects. I was the Software Lead of the project, using Jira to delegate tasks and ensuring communication between the designers and programmers on the team. Within the code, I was responsible for ensuring the app could scale across all device sizes, as well as coding a large part of the users personal profile and history.

It is coming to the App Store soon!

Online Scrabble

Tools and Skills: OCaml, OCaml graphics

This game mimics online Scrabble, in which players can pass a computer around to play. It was coded in Ocaml, a functional programming language, and displayed using OCaml graphics. I worked with two others on this project. I coded the functionality of the player tiles, ie how the user clicks and drags them, as well as how each player's tiles are stored in the backend. The game also has an automated scoring system, which I contributed a large part of.
